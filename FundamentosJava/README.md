# ☕ Fundamentos Java

Este repositório reúne meus estudos iniciais em **Java**, com foco em sintaxe básica, lógica de programação e introdução à Programação Orientada a Objetos (POO).

---

## 📚 Conteúdo do repositório

### 🧠 Sintaxe básica
Exercícios iniciais para aprendizado da linguagem Java, incluindo:

- Variáveis e tipos de dados
- Estruturas condicionais (`if`, `switch`)
- Estruturas de repetição (`for`, `while`, `do-while`)
- Entrada de dados com `Scanner`

---

### 🎮 Projeto RPG (POO)

Projeto simples simulando um sistema de RPG, com foco em conceitos de:

- Classes e objetos
- Encapsulamento
- Métodos
- Interação entre objetos

Exemplo de funcionalidades:
- Cadastro de personagem
- Atributos como nome, poder e energia
- Sistema de ataque
- Habilidades especiais
- Aumento de energia

---

### 🧩 Desafios

Exercícios adicionais para reforço de lógica de programação e prática da linguagem.
