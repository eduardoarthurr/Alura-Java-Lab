# 📚 Estudos Java, Spring Boot e Docker - Alura

Este repositório foi criado para armazenar os conteúdos, exercícios, projetos e anotações desenvolvidos durante os cursos da Alura relacionados a **Java**, **Spring Boot** e **Docker**.

## 🎯 Objetivo

Centralizar todo o material de estudo em um único local, facilitando:

- Revisão dos conteúdos aprendidos;
- Consulta de exemplos práticos;
- Acompanhamento da evolução nos estudos;
- Construção de um portfólio de projetos e exercícios.

## 🚀 Tecnologias Estudadas

### Java
- Sintaxe básica
- Orientação a Objetos
- Collections
- Streams
- Tratamento de Exceções
- Boas práticas de desenvolvimento

### Spring Boot
- Criação de APIs REST
- Consumo de APIs
- Spring Data JPA
- Hibernate
- Validação de dados
- Testes automatizados
- Segurança com Spring Security

### Docker
- Containers
- Imagens
- Dockerfile
- Docker Compose
- Persistência de dados
- Redes Docker
